Satellite Image Classifier

This project is based on multispectral satellite image interpretation. It uses machine learning to analyze different land cover types such as water bodies, vegetation, deserts, and cloudy regions.

The classifier predicts the type of area from the uploaded satellite image using trained deep learning models.
